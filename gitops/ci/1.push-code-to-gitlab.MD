## Create a project on Gitlab and Clone to local workspace

```shell
# cd ~/projects/
# git clone ssh://git@gitlab.gitops.com:32600/root/demo-app.git 
# 32600 is nginx controller NodePort
# cd demo-app

```

## Copy demo code to workspace
```shell
# cp -r demo-app/* .
```

## Push code to Gitlab
```shell
# git add .
# git commit -m "init commit"


```
